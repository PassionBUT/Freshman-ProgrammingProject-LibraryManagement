# 📚 图书馆管理系统

一个基于 C++ 的控制台图书馆管理系统，支持管理员、读者、游客三种角色，实现图书管理、用户管理、借阅管理、搜索推荐、操作日志等完整功能。

## ✨ 功能特性

### 👑 管理员功能
- 图书管理：添加、删除、更新图书，支持 CSV 批量导入
- 用户管理：添加、删除、更新用户，重置密码，注销/恢复用户
- 借阅记录查询：按账号、ISBN、日期范围、记录号搜索
- 管理员操作日志：自动记录所有管理员操作，支持按管理员、操作类型、日期范围检索

### 👤 读者功能
- 图书搜索：ISBN 精确搜索，书名/作者/出版社模糊搜索
- 借书/还书：自动计算应还日期和逾期天数
- 个人借阅记录：查看自己的借阅历史
- 排行榜：借阅次数排行榜、最新图书排行榜、读者借阅排行榜
- 个性化推荐：基于协同过滤的图书推荐

### 👀 游客模式
- 只读访问：可搜索图书、查看排行榜，无需登录

### 🛠️ 其他特性
- 数据持久化：图书、用户、借阅记录、管理员日志自动保存到文件
- 分页显示：所有列表支持分页浏览（每页10条）
- 异常处理：输入验证、文件读写异常捕获

## 🖥️ 技术栈

- **语言**：C++17
- **核心特性**：智能指针（`std::shared_ptr`）、STL容器（`unordered_map`、`map`、`vector`）、泛型编程（模板函数）
- **文件存储**：纯文本文件（CSV 格式），自定义序列化/反序列化
- **搜索算法**：倒排索引（ISBN、书名、作者、出版社）
- **推荐算法**：基于物品的协同过滤（共现矩阵）
- **日期处理**：`<ctime>` 获取系统时间，自定义日期比较

## 🚀 如何运行

### 环境要求
- 操作系统：Windows（使用 `system("cls")` 清屏）
- 编译器：支持 C++17 的编译器（如 GCC 7+、MSVC 2017+、MinGW）
- 构建工具：无需额外依赖，直接编译所有 `.cpp` 文件

### 编译与运行

#### 使用 g++（MinGW / Linux）

```bash
# 进入项目根目录（包含 src 文件夹和 data 文件夹）
g++ -std=c++17 -o library.exe *.cpp
./library.exe
```

## 🧩 项目亮点

- 协同过滤推荐：基于用户历史借阅记录，计算图书共现矩阵，推荐相关图书。
- 倒排索引搜索：为书名、作者、出版社建立索引，模糊搜索响应迅速。
- 完整的管理员审计日志：每个关键操作都被记录，可按多种条件查询。
- 分页模板函数：通过泛型实现任意类型数据的分页输出，代码复用率高。
- 智能指针管理：全部使用 std::shared_ptr，避免内存泄漏。
